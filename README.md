# 🧮 Calculator-website

## 📌 Overview
This is a simple and responsive calculator built using HTML, CSS, and JavaScript.

## 🚀 Features
- Addition, Subtraction, Multiplication, Division
- Clear (C) Button
- Delete (DEL) Button
- Responsive Design

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript
